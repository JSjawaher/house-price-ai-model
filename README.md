# House Price Classification Model 🏠🤖

## 📌 وصف المشروع
مشروع ذكاء اصطناعي يهدف إلى بناء نموذج تعلم عميق باستخدام Keras لتصنيف أسعار المنازل بناءً على مجموعة من الخصائص (Binary Classification).

---

## 🧠 التقنيات المستخدمة
- Python  
- Keras  
- TensorFlow  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

---

## 📊 خطوات العمل
- تحميل البيانات (CSV File)  
- تقسيم البيانات إلى تدريب واختبار  
- معالجة البيانات وتوحيد القيم (Standardization)  
- بناء نموذج Neural Network باستخدام Keras  
- تدريب النموذج وتحسين الأداء  
- تقييم النموذج باستخدام Accuracy و Loss  

---

## 🧠 نموذج الذكاء الاصطناعي
تم بناء نموذج يحتوي على عدة طبقات Dense مع تفعيل ReLU و Sigmoid للتصنيف الثنائي.

---

## 📈 النتائج
تم تقييم أداء النموذج باستخدام:
- دقة النموذج (Accuracy)  
- قيمة الخطأ (Loss)  
- منحنيات التدريب والاختبار  

---
